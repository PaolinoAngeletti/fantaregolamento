# Fanta Regolamento

## Public URL

https://paolinoangeletti.github.io/fantaregolamento/

## How to Run

To run this application, just launch the index.html file.  
You can do it via a random server (e.g. the one in Visual Code - Go Live) or by starting via IDE in localhost.

## How to Test

To run the tests, again, simply run the index.html file in the tests' folder.  
They were developed using the Jasmine framework.
